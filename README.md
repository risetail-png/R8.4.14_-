# 橋梁点検PDF リンク追加ツール

橋梁定期点検PDFの「損傷図（その９）」と「損傷写真（その１０）」の間に
ナビゲーションボタンを自動追加するデスクトップアプリです。

**v7より複数径間PDF（1径間・2径間など）に対応しました。**

## ダウンロード（ビルド済みアプリ）

右側の **Actions** タブ → 最新のワークフロー → **Artifacts** から
お使いのOS向けファイルをダウンロードしてください。

| OS | ファイル名 |
|---|---|
| Windows | `橋梁点検PDFリンク追加ツール-Windows.zip` |
| macOS | `橋梁点検PDFリンク追加ツール-Mac.zip` |

## 使い方

1. ダウンロードした `.exe`（Windows）または実行ファイル（Mac）を起動
2. PDFをドラッグ＆ドロップ
3. 「▶ 処理開始」をクリック

## 対応PDF

- 単一径間PDF（従来通り）
- 複数径間PDF（v7新機能）
  - ヘッダーの径間番号を自動検出
  - 損傷図の写真番号（例: `写真番号1-1`）と損傷写真を径間ごとに正確にリンク
